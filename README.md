<h1>Отчет по эффективности сотрудников и отделов 📊</h1>

<p>
Инструмент для анализа эффективности работы сотрудников и отделов в <b>Bitrix24</b>.<br>
Позволяет отслеживать статус сделок и оценивать результативность команды за выбранный период.
</p>

<h2>🔹 Проблема</h2>

<p>Клиент столкнулся с необходимостью:</p>

<ul>
<li>Оценивать эффективность сотрудников и отделов</li>
<li>Понимать, сколько сделок находится в работе, успешно завершено или провалено</li>
<li>Анализировать результаты команды за выбранный период</li>
</ul>

<hr>

<h2>🔹 Решение</h2>

<p>Разработал приложение, которое позволяет:</p>

<ul>
<li>Собирать аналитику по сделкам в разрезе сотрудников или отделов</li>
<li>Разделять сделки по статусам (в работе, успешные, проваленные)</li>
<li>Оценивать эффективность работы команды</li>
<li>Гибко настраивать фильтры и режим отображения данных</li>
</ul>

<hr>

<h2>🔹 Показатели отчета</h2>

<table>
<tr>
<th>Показатель</th>
<th>Описание</th>
</tr>
<tr>
<td>Сотрудник / Отдел</td>
<td>Ответственный сотрудник или отдел</td>
</tr>
<tr>
<td>Всего сделок</td>
<td>Общее количество сделок</td>
</tr>
<tr>
<td>В работе</td>
<td>Сделки в процессе</td>
</tr>
<tr>
<td>Заключенные</td>
<td>Успешно завершенные сделки</td>
</tr>
<tr>
<td>Проваленные</td>
<td>Неуспешные сделки</td>
</tr>
</table>

<p><b>Фильтры:</b></p>
<ul>
<li>Статус сделки</li>
<li>Режим отображения (по сотрудникам / по отделам)</li>
<li>Период (по дате создания сделки)</li>
</ul>
